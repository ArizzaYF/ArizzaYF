## 🤖 AI-Driven Workflow

I don't use AI to generate code I don't understand. I use it as a **senior collaborator that's always available**.

```
My AI workflow in practice:
 ├── 🧠  Architectural reasoning  →  rubber-duck with LLMs before writing a single line
 ├── 📄  Documentation            →  first draft in seconds, refined with context
 ├── 🐛  Debugging                →  pattern recognition at scale, not Stack Overflow roulette
 └── ⚡  Code review              →  AI flags edge cases I missed; I decide what matters
```

> *"The developer who masters AI tooling doesn't write less code — they write better code, faster."*
